# Contributing to RollScape

Thank you for your interest in RollScape!

## Current Status

RollScape is currently in **early development** and is **not accepting external contributions** at this time.

## Future Contributions

We plan to open source certain components of RollScape in the future, including:

- Community homebrew content
- Asset packs
- Rule system modules
- Documentation improvements

We'll update this file when we're ready to accept contributions.

## Reporting Issues

If you're a tester or early user and encounter bugs:

1. Check if the issue already exists
2. Provide detailed reproduction steps
3. Include system information
4. Add screenshots if applicable

## Feature Requests

Feature requests are welcome! Please:

1. Search existing requests first
2. Describe the use case
3. Explain why it would benefit the community
4. Consider alternative solutions

## Questions?

For questions about RollScape, please see our documentation in the `/docs` folder.
